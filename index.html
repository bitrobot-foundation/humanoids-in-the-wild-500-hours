Humanoid In Homes

An in-the-wild household dataset
for whole-body humanoid robot learning.

Overview

Built for learning from real homes, not lab-only scenes.

Humanoid-In-Homes focuses on task execution in natural household environments where layout, object state, lighting, clutter, and human operating style vary from episode to episode.

The dataset is collected through human whole-body teleoperation of Unitree G1, providing demonstrations for mobile manipulation, bimanual interaction, and long-horizon household skills.

Dataset

What data is collected?

Each episode records human whole-body teleoperation of Unitree G1 in real homes, combining camera streams, robot states, action traces, and language annotations.
